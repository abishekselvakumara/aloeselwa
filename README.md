## Aloe Vera Atelier — Landing Page

Premium, editorial-style landing page for a global aloe vera company, built with **React**, **Vite**, and **Tailwind CSS**.

### Tech stack

- **React 18** + **Vite**
- **Tailwind CSS 3** with custom aloe color palette
- Minimal custom IntersectionObserver hook for text reveal animations

### Getting started

1. Install dependencies:

```bash
npm install   # or: npm install / yarn install
```

2. Run the development server:

```bash
npm run dev       # or: npm run dev / yarn dev
```

3. Open the URL printed in your terminal (typically `http://localhost:5173`) in your browser.

### Customization

- Update editorial copy and brand naming in `src/App.jsx`.
- Adjust typography and spacing via Tailwind classes or the custom fonts defined in:
  - `index.html` (Google Fonts)
  - `tailwind.config.cjs` and `src/styles.css`.
